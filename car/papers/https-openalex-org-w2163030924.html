A statistical approach to 3d object detection applied to faces and cars

Year: 2000

Primary subgenre: computer vision

Secondary subgenres: object detectionvehicle license plate recognition

Summary: In this thesis, we describe a statistical method for 3D object detection. In this method, we decompose the 3D geometry of each object into a small number of viewpoints. For each view-point, we construct a decision rule that determines if the object is present at that specific orienta-tion.
