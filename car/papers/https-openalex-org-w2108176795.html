Language-based multimodal displays for the handover of control in autonomous cars

Year: 2015

Primary subgenre: context

Secondary subgenres: computer vision

Summary: This paper presents a first evaluation of multimodal language-based warnings for handovers of control in autonomous cars. A set of possible handover situations varying in urgency is described. A set of multimodal, language-based warnings for these situations is then introduced.
