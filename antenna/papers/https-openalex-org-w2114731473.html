Multiuser MIMO in Distributed Antenna Systems With Out-of-Cell Interference

Year: 2011  DOI ↗

Main topic: antenna system

Abstract
Distributed antenna systems (DAS) augment the base station's transmit capability by adding multiple remote radio units, connected to the base station via a high bandwidth and low latency link. With DAS, the base station operates as if it had multiple antennas, but the antennas happen to be in different geographic locations.
