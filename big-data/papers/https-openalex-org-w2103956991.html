Quantum Support Vector Machine for Big Data Classification

Year: 2014

Primary subgenre: algorithm

Secondary subgenres: artificial intelligence

Summary: Supervised machine learning is the classification of new data based on already classified training examples. In this work, we show that the support vector machine, an optimized binary classifier, can be implemented on a quantum computer, with complexity logarithmic in the size of the vectors and the number of training examples.
