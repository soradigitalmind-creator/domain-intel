Maximally localized generalized Wannier functions for composite energy bands

Year: 1997

Main topic: renewable energy

Related topics:

Summary: We discuss a method for determining the optimally localized set of generalized Wannier functions associated with a set of Bloch bands in a crystalline solid. By ``generalized Wannier functions'' we mean a set of localized orthonormal orbitals spanning the same space as the specified set of Bloch bands.
