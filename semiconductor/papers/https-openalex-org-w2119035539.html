Experimental Observation of the Spin-Hall Effect in a Two-Dimensional Spin-Orbit Coupled Semiconductor System

Year: 2005

Primary subgenre: hall effect

Secondary subgenres: magnetic fieldspinspin polarization

Summary: We report the experimental observation of the spin-Hall effect in a 2D hole system with spin-orbit coupling. The 2D hole layer is a part of a p-n junction light-emitting diode with a specially designed coplanar geometry which allows an angle-resolved polarization detection at opposite edges of the 2D hole system.
