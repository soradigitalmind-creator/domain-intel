ON THE TRANSITION TO METALLIC CONDUCTION IN SEMICONDUCTORS

Year: 1956

Primary subgenre: compound semiconductor

Secondary subgenres:

Summary: A discussion is given of the conductivity to be expected from a crystalline array of atoms when the interatomic distance a is varied. It is shown that both for monovalent atoms and divalent atoms the conductivity is zero at T = 0 when a is large, the wave function of the system being real so that no current is possible. In both types of atom a discontinuous transition to a state showing metallic c
