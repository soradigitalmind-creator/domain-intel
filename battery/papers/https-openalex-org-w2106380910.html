A Generic Battery Model for the Dynamic Simulation of Hybrid Electric Vehicles

Year: 2007

Primary subgenre: advanced battery technologies research

Secondary subgenres: automotive engineeringcomputer scienceelectrical engineeringengineering

Summary: This paper presents an easy-to-use battery model applied to dynamic simulation software. The simulation model uses only the battery State-Of-Charge (SOC) as a state variable in order to avoid the algebraic loop problem.
